# How to run the project

1. Open terminal
2. Go to the project folder
3. `npm i` in the terminal
4. After it is all done, `npm start` in terminal
5. It should automatically open the app in the browser. If not, go to http://localhost:3000 in you browser

# How to run unit test
1. Assuming you have installed everything and is running the app, open new terminal
2. Go to the project folder
3. `npm test` in the terminal
4. Type `a` in the terminal and then click `ENTER` to run all the tests
